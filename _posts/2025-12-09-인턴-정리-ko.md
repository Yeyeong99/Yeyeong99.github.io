# 문제 상황
- 어제와 같은 파일을 실행했는데 갑자기 환경 변수가 None으로 출력됨.
- config.py 내에선 제대로 로드 됨

# 가능한 원인
## 1. load_dotenv 전에 API_KEY 할당이 이루어짐
- load_dotenv()를 먼저 호출해야 환경 변수로 값이 제대로 할당됨
### 해결책) 함수 활용
1. 기존 방식
``` pypython
import os
from dotenv import load_dotenv

load_dotenv()

API_KEY = os.environ.get("API_KEY")
```

2. 함수 활용
``` pypython
import os
from dotenv import load_dotenv

load_dotenv()

def get_api_key():
    return os.environ.get("API_KEY")
```
- 함수로 선언할 경우 다른 페이지에서 사용 시 무조건 load_dotenv 보다 늦게 호출되기 때문에 오류 방지 가능

## 2. 파이썬 실행 방법에 따른 오류
- `python -m tests.total.py`로 실행해서 생긴 문제
### 해결)
- .py 떼고 실행
